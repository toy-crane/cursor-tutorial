# Cursor Tutorial

이 프로젝트는 [Next.js](https://nextjs.org)를 사용하여 만들어진 간단한 Hello
World 애플리케이션입니다.

## 시작하기

개발 서버를 실행하려면:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인할
수 있습니다.

페이지를 수정하려면 `src/app/page.tsx` 파일을 편집하면 됩니다. 파일을 수정하면페
이지가 자동으로 업데이트됩니다.

## 기술 스택

- [Next.js](https://nextjs.org) - React 기반의 웹 프레임워크
- [Tailwind CSS](https://tailwindcss.com) - 유틸리티 기반 CSS 프레임워크

## 배포하기

Next.js 앱을 배포하는 가장 쉬운 방법은 Next.js의 제작자인
[Vercel Platform](https://vercel.com)을 사용하는 것입니다.

더 자세한 내용은
[Next.js 배포 문서](https://nextjs.org/docs/app/building-your-application/deploying)를
참조하세요.
