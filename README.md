# Cursor Tutorial - Next.js Project

이 프로젝트는 [Next.js](https://nextjs.org)와
[Tailwind CSS](https://tailwindcss.com)를 사용하여 만들어진 간단한 웹 애플리케이
션입니다.

## 프로젝트 시작하기

개발 환경에서 프로젝트를 실행하려면 다음 명령어를 사용하세요:

```bash
npm run dev
# 또는
yarn dev
# 또는
pnpm dev
# 또는
bun dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)으로 접속하여 결과를
확인할 수 있습니다.

애플리케이션의 메인 페이지는 `src/app/page.tsx` 파일에서 수정할 수 있습니다. 파
일을 저장하면 변경 사항이 자동으로 반영됩니다.

## 주요 기술 스택

- **프레임워크**: [Next.js](https://nextjs.org) (App Router 기반)
- **스타일링**: [Tailwind CSS](https://tailwindcss.com)
- **UI 컴포넌트**: (만약 사용한다면 여기에 명시, 예:
  [shadcn/ui](https://ui.shadcn.com/))
- **타입스크립트**: [TypeScript](https://www.typescriptlang.org/)
