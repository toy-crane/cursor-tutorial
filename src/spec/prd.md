# WIFI QR 코드 카드 생성 서비스 요구사항 명세 (PRD)

## 1. 개요

매장에서 고객에게 WIFI 접속 정보를 쉽게 안내하기 위해, QR 코드가 포함된 카드를생
성하고 이미지로 다운로드할 수 있는 서비스를 개발한다. 사용자는 브랜드 이름, 네트
워크 이름(SSID), 비밀번호, 카드 배경색을 입력하여 맞춤형 WIFI 안내 카드를 제작할
수 있다.

## 2. 목표

- 사용자가 쉽고 빠르게 WIFI 접속 안내 카드를 생성할 수 있도록 한다.
- 미니멀하고 깔끔한 디자인의 카드를 제공한다.
- 생성된 카드를 이미지 파일로 다운로드하여 오프라인 매장에서 활용할 수 있도록 지
  원한다.

## 3. 주요 기능

1.  **WIFI 정보 입력:**
    - 브랜드 이름 입력 필드
    - 네트워크 이름(SSID) 입력 필드
    - 비밀번호 입력 필드
    - 카드 배경색 선택 (컬러 피커 또는 사전 정의된 색상 팔레트)
2.  **QR 코드 생성:**
    - 입력된 네트워크 이름(SSID)과 비밀번호를 기반으로 WIFI 접속용 QR 코드를 동
      적으로 생성한다. (WIFI:S:<SSID>;T:<WPA|WEP|>;P:<PASSWORD>;;)
3.  **카드 디자인 커스터마이징:**
    - 사용자가 선택한 배경색을 카드 전체 배경에 적용한다.
4.  **카드 미리보기:**
    - 사용자가 입력한 정보와 선택한 배경색이 적용된 카드 디자인을 실시간으로 미
      리 보여준다.
5.  **이미지 다운로드:**
    - 생성된 WIFI QR 코드 카드를 이미지 파일(예: PNG, JPG) 형태로 다운로드하는기
      능을 제공한다.

## 4. 카드 레이아웃 및 디자인

- **디자인 컨셉:** 미니멀리즘
- **구성 요소:**
  - **상단 중앙:** "WIFI 접속" 텍스트 제목
  - **중앙:** 생성된 WIFI QR 코드 이미지
  - **하단 중앙:** 입력된 "브랜드 이름" 텍스트
  - **하단 오른쪽 구석:** 작은 크기의 "by toycrane" 레이블 텍스트

## 5. 비기능적 요구사항

- 웹 기반 서비스로 구현한다.
- 반응형 디자인을 고려하여 다양한 디바이스에서 사용 가능하도록 한다. (선택 사항)

## 6. 기술 스택 (Tech Stack)

- **Framework:** Next.js (React 기반 프레임워크, SSR/SSG 및 라우팅 기능 내장)
- **Form Management:** `react-hook-form` (폼 상태 관리 및 유효성 검사)
- **QR Code Generation:** `qrcode.react` 또는 유사 JavaScript 라이브러리 (클라이
  언트 측에서 QR 코드 동적 생성)
- **Styling:** Tailwind CSS + Shadcn/ui (미리 정의된 컴포넌트와 디자인 시스템을
  활용한 빠른 UI 개발)
- **Image Generation/Download:** HTML Canvas API 또는 `html2canvas` 라이브러리
  (DOM 요소를 이미지로 변환하여 다운로드)
- **Deployment:** Vercel (Next.js 최적화 배포 플랫폼)
